# 课堂智能监控系统

基于深度学习的课堂实时监控系统，支持人脸识别、行为检测和考勤管理。

## 功能特性

### 核心功能
- **人脸识别**：基于 YOLOv8-face + ArcFace 的实时人脸检测与身份识别
- **行为检测**：检测学生睡觉、使用手机等违规行为
- **考勤管理**：自动记录学生出勤情况，生成考勤报告
- **多源输入**：支持本地摄像头、网络视频流(RTSP/RTMP/HTTP)、本地视频文件

### 技术亮点
- **人脸对齐**：基于关键点的自适应人脸对齐，提升识别准确率
- **头部姿态检测**：识别抬头、低头、正常姿态，适应不同视角
- **图像增强**：
  - 本地摄像头：轻量级 CLAHE 对比度增强
  - 网络视频流：完整增强（去噪、锐化、多帧融合）
- **身份缓存**：基于轨迹的身份绑定，识别成功后无需重复识别
- **智能扫描**：自动扫描局域网内的 RTSP/RTMP/HTTP 视频流

## 系统要求

- Python 3.8+
- Windows 10/11
- CUDA 11.x（可选，用于 GPU 加速）

## 安装

### 1. 克隆项目
```bash
git clone <repository_url>
cd pcrty
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 下载模型文件

首次运行时，程序会自动下载以下模型：
- `yolov8n-face.pt` - 人脸检测模型
- `yolov8n-pose.pt` - 姿态估计模型
- `yolov8n.pt` - 通用目标检测模型

### 4. 准备学生数据库

在 `photos_dir` 目录下为每位学生创建以姓名命名的文件夹，放入该学生的照片：

```
photos_dir/
├── 张三/
│   ├── 照片1.jpg
│   ├── 照片2.png
│   └── ...
├── 李四/
│   ├── 照片1.jpg
│   └── ...
└── ...
```

然后运行：

```bash
python db_generate.py
```

这将生成 `student_db.npy` 人脸特征数据库文件。

**注意**：
- 照片要求正面清晰，单人人脸
- 支持增量更新，已有学生不会重复处理
- 每张照片会自动生成镜像增强特征
- 同一学生文件夹内的多张照片会被合并处理

## 使用方法

### 启动程序
```bash
python monitor_GUI.py
```

### 界面操作

1. **选择输入源类型**
   - 摄像头：输入摄像头索引（如 0、1）
   - RTSP/RTMP：输入网络视频流地址
   - 本地视频：点击"浏览"选择视频文件

2. **智能扫描**
   - 点击"扫描设备"自动发现局域网内的视频流
   - 从列表中选择要连接的流

3. **开始监控**
   - 点击"应用/切换"连接视频源
   - 系统自动进行人脸识别和行为检测

4. **查看报告**
   - 点击"生成报告"导出考勤和行为记录

## 核心算法

### 人脸识别流程
1. YOLOv8-face 检测人脸及 5 点关键点
2. 基于关键点进行人脸对齐
3. 图像增强（根据输入源类型选择增强级别）
4. ArcFace 提取 512 维特征向量
5. 与数据库进行余弦相似度匹配
6. 身份绑定与缓存

### 行为检测
- **睡觉检测**：基于姿态关键点，判断眼睛是否低于肩膀
- **手机检测**：基于 YOLO 目标检测，识别手机类别

## 配置说明

### 识别阈值调整

在 `ClassroomFinalSystem` 类中可调整以下参数：

```python
# 人脸检测置信度
face_conf = 0.30  # 本地摄像头
face_conf = 0.35  # 网络视频流

# 识别间隔（帧数）
recognize_interval = 10   # 本地摄像头
recognize_interval = 20   # 网络视频流

# 相似度阈值
sim_threshold = 0.40
```

### 网络扫描配置

在 `scan_all_streams()` 函数中可配置扫描范围：
```python
target_network = "192.168.1.0/24"  # 目标网段
scan_ports = [554, 8554, 1935, 8080]  # 扫描端口
```

## 常见问题

### Q: 识别速度慢怎么办？
A: 
- 确保使用 GPU 加速（安装 CUDA 版本的 PyTorch）
- 降低识别间隔帧数
- 使用本地摄像头时自动使用简化增强模式

### Q: 识别准确率低？
A:
- 确保学生照片质量良好、正面清晰
- 增加每个学生的训练照片数量
- 调整相似度阈值

### Q: 无法连接网络视频流？
A:
- 检查网络连接和视频流地址
- 确认视频流协议（RTSP/RTMP/HTTP）
- 使用"扫描设备"功能自动发现
